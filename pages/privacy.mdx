# Privacy Policy

## Open Source Project

PromptX is an open-source project licensed under the MIT License. As a static documentation website and desktop application, we prioritize user privacy and transparency.

## Data Collection

### Website (promptx.deepractice.ai)

This website is a static site hosted on Cloudflare Pages. We do not:
- Collect personal information
- Use cookies for tracking
- Store user data on our servers
- Run analytics or tracking scripts

**Cloudflare Pages** may collect standard server logs (IP addresses, user agents) for infrastructure purposes. Please refer to [Cloudflare's Privacy Policy](https://www.cloudflare.com/privacypolicy/) for details.

### Desktop Application

The PromptX desktop application:
- Runs entirely on your local machine
- Does not collect or transmit user data to our servers
- Stores all configuration and data locally
- Only communicates with Claude AI when you explicitly use AI features

When using AI features, your prompts are sent to Anthropic's Claude API according to your API configuration. Please refer to [Anthropic's Privacy Policy](https://www.anthropic.com/legal/privacy) for how they handle your data.

## Open Source Transparency

As an open-source project, you can:
- Review our [source code on GitHub](https://github.com/Deepractice/PromptX)
- Verify what data the application accesses
- Build the application yourself from source
- Report privacy concerns via GitHub Issues

## Third-Party Services

PromptX integrates with:
- **Claude AI (Anthropic)** - AI language model API
- **MCP Servers** - Model Context Protocol servers you configure

When using these services, their respective privacy policies apply:
- [Anthropic Privacy Policy](https://www.anthropic.com/legal/privacy)
- [MCP Protocol Documentation](https://modelcontextprotocol.io)

## Changes to This Policy

We may update this privacy policy as the project evolves. Changes will be announced via:
- GitHub repository updates
- Release notes

## Contact

For privacy-related questions or concerns:

- Visit [Deepractice](https://deepractice.ai) for general inquiries
